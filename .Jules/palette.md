## 2025-05-14 - Accessible Transit Countdown Patterns
**Learning:** Standard status colors like #27ae60 (green) and #3498db (blue) often fail WCAG AA contrast requirements on white backgrounds or when used as decorative indicators. Users also benefit significantly from semantic list structures (UL/LI) for grouped time data, which provides better context for screen readers.
**Action:** Always verify contrast ratios for status indicators and use darker, more accessible shades (#1b5e20, #1565c0) by default. Prioritize semantic HTML for all lists of departures.
